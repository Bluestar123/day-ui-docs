# 介绍

day-ui 是基于 vue3 实现的组件库，样式采用的 element-plus，用于学习交流。

项目发布到了 npm 上，可以下载使用，但是还没有实现项目引用 babel-plugin-import 实现按需加载功能。只能全部引入

# 特点

  • 项目使用 js 开发， ts 定义类型

# 安装

npm i day-ui -S

# 使用

import DayUI from 'day-ui'
import 'day-ui/lib/style.css'
